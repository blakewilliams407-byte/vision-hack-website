# ShowUp: Event RSVP Platform

Welcome to ShowUp, a modern and stylish event RSVP platform built with Next.js and Firebase Studio.

## About The Project

ShowUp provides a seamless experience for both event hosts and guests.

- **For Guests:** A beautiful, single-page view to see event details and RSVP instantly.
- **For Hosts:** A powerful dashboard to manage event details, track the guest list, and control RSVP status in real-time.

This application is built with:
- [Next.js](https://nextjs.org/) (App Router)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [ShadCN UI](https://ui.shadcn.com/) for components
- [Genkit](https://firebase.google.com/docs/genkit) for AI-powered features (like the "Vibe Generator")

## Getting Started

- The main guest-facing page is located at `src/app/page.tsx`.
- The host dashboard can be found at `src/app/dashboard/page.tsx`.
- Application state is managed centrally in `src/app/state/app-context.tsx`.

Feel free to explore and customize your event!
