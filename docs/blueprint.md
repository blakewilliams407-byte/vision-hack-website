# **App Name**: ShowUp

## Core Features:

- Event Page Display (Guest View): Displays event name, date, time, venue, vibe description, and a cover photo placeholder to guests.
- Guest RSVP Submission: Allows guests to enter their first name to RSVP for an event and receive a confirmation.
- Dynamic RSVP Status: Shows guests the current number of people who have RSVP'd, or a 'RSVPs Closed' message if applicable.
- Host Event Details Editor: Host can view and inline edit essential event details like title, date, and venue through a clean interface.
- Live Guest List & Headcount: Displays a real-time list of RSVP'd guests, including their names and RSVP time, alongside a running headcount.
- RSVP Availability Toggle: Host can easily open or close RSVPs with a toggle, immediately updating the guest-facing page.
- Vibe Description Generator Tool: Assists the host in generating a captivating and creative 'vibe description' for the event using generative AI.

## Style Guidelines:

- Primary color: Electric violet (#6600FF) to convey boldness and energy.
- Background color: A near-black with a hint of violet (#0E0B0F) for a sophisticated, dark aesthetic.
- Accent color: Hot pink (#F21AC1) for high-contrast elements and interactive components.
- Headlines and prominent text: 'Space Grotesk' (sans-serif) for a modern, techy, and bold feel, reinforcing the 'hype' theme.
- Body text and functional elements: 'Inter' (sans-serif) for legibility and a clean, objective command center look on the dashboard.
- Utilize simple, high-contrast vector icons that align with the electric and modern aesthetic, focusing on clarity and immediate recognition.
- Mobile-first design for all views; Guest page features immersive full-width sections and prominent calls-to-action, while the Host dashboard adopts a structured, minimalist grid layout for efficient data management.
- Subtle yet responsive UI feedback with animations on button clicks, toggle states, and RSVP confirmations, contributing to a dynamic user experience without being distracting.
